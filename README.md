This project is a C++ implementation of the Traveling Salesman Problem using multiple graph algorithms and representations.

The program:

Loads graph data from a text file
Stores the graph using:
Adjacency Matrix (Static Representation)
Adjacency List (Dynamic Representation)
Solves the TSP using:
Brute Force Algorithm
Greedy (Nearest Neighbor) Algorithm
Finds shortest paths using:
Dijkstra’s Algorithm
Compares algorithm performance and route costs
